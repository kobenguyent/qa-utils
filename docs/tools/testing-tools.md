Generate CI/CD workflow configurations for multiple platforms.

- **Platforms:** GitHub Actions, GitLab CI, Azure DevOps, Jenkins, Bitbucket Pipelines
- **Test Types:** API testing, E2E testing
- **Runners:** Playwright, Puppeteer, WebDriverIO, Cypress
- **Node.js:** 16, 18, 20, 21 support
- NPM publishing workflows
- Real-time preview with syntax highlighting
- One-click download

**Route:** `/workflow-generator`
